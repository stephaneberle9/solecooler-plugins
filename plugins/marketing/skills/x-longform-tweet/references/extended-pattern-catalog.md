# Extended Pattern Catalog — Patterns 3, 5, 6, 9

Top 5 patterns (1, 2, 4, 7, 8) live inline in SKILL.md. The four less-frequently-used patterns live here so SKILL.md stays scannable. Use these when the topic shape calls for them — every pattern in here is fully production-ready, none are deprecated.

---

## Pattern 3: The Bookend Reframe

**Family:** Strategic (action-oriented, opportunity-focused)

**Best for:** making simple math or logic feel like a revelation

```
[Opening phrase]

[Line 1 — simple fact]

[Line 2 — builds on it]

[Line 3 — builds further]

...

[Punchline question]

[Same opening phrase repeated]
```

**Rules:**
- One idea per line. Each builds on the previous.
- Let the math or logic speak. No editorializing between steps.
- The repetition of the opening line IS the emphasis. No bold, no caps needed.
- The question near the end does all the persuading.

**Length range:** 400-700 chars

## Pattern 5: The Paradox Reveal

**Family:** Commentary (philosophical, cultural analysis)

**Best for:** counterintuitive takes, "here's what nobody sees" posts

```
[Quote or paraphrase the naive/common response]

[Dismiss it — can be blunt]

[Explain the actual dynamic at play — the forces underneath]

[Punchline — often uncomfortable, no softening]
```

**Rules:**
- Open by putting the opposing view in someone else's mouth, then dismantle it.
- Casual dismissal works as a transition ("lol. it's not fixable.")
- Use frameworks from unexpected fields (physics, biology, game theory) naturally
- End hard. No "but there's hope" unless you mean it.

**Length range:** 400-700 chars

## Pattern 6: The Extended Analogy

**Family:** Commentary (philosophical, cultural analysis)

**Best for:** explaining dynamics, comparing platforms/tools/approaches

```
[Set up the metaphor domain]

[X is like A: [1-sentence explanation]]
[Y is like B: [1-sentence explanation]]
[Z is like C: [1-sentence explanation]]
...

[Close with why this framing matters — 1 sentence]
```

**Rules:**
- Single metaphor domain extended across 3-6 items
- Each item gets exactly one line. Tight, parallel.
- One item should be the standout — slightly mean, funny, or unexpected
- Close is one sentence, not a paragraph

**Length range:** 400-600 chars

## Pattern 9: The Personal Honesty Drop

**Family:** Builder (raw, honest, in-progress)

**Best for:** life moments, vulnerable observations, things you normally wouldn't post

```
[Surprisingly honest statement]

[Context that makes it relatable — not a life lesson, just the situation]

[Observation — what you noticed, not what it "taught" you]
```

**Rules:**
- No moral at the end. Observations > lessons.
- Internet-native language is fine (lmao, wtf, tbh)
- Shorter is better. Vulnerability doesn't need a thousand words.

**Length range:** 200-500 chars
